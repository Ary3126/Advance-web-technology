# Student Portfolio React App

This project is a Vite + React implementation of a student portfolio page for Practical 1. It uses reusable functional components such as Header, About, Skills, Footer, and NavBar, with props passed from App.jsx to render dynamic content.

## Features

- Responsive single-page portfolio layout
- Reusable components with independent structure
- Props used to pass data into multiple components
- Custom CSS styling for a modern student portfolio look

## Run Locally

1. Open the project folder
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open the local URL shown by Vite in your browser

## Build for Production

```bash
npm run build
```
